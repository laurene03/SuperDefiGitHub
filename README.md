# SuperDefiGitHub

## Description
Ce projet consiste à réaliser un site internet pour un concours de photos ornithologiques dont le sujet est l’espace naturel de la motte.

## Plan du site 
- Page *Accueil* présentant l'espace naturel de la motte
- Page *Concours de photos* avec un formulaire de participation et un formulaire de vote
